Detailed Description

template<typename T>
class Dataset< T >

This class permits to create complex datasets and use them to feed the network.

This class provides two containers for Input and Target datas. The Dataset can be created using the Parser directly from a CSV (Comma-Separeted Values) file or an XML file. During the backpropagation training it loads Input and Output values and use them to feed the Network.
